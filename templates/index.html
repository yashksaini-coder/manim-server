Manim AI API
POST /v1/render/video
Render a video using Manim based on user-provided code and parameters.
Body Example:
{
"code": "from manim import *\nclass GenScene(Scene):\n    def construct(self):\n        c = Circle()\n        self.play(Create(c))",
"file_name": "scene.py",
"file_class": "GenScene",
"stream": false,
"user_id": "1234567890",
"project_name": "My Project",
"iteration": 1,
"aspect_ratio": "16:9"
}
Test with curl:
curl -X POST http://localhost:5000/v1/render/video -H "Content-Type: application/json" -d @body.json
POST /v1/generate/code
Generate Manim code using an LLM (OpenAI or Groq).
Body Example:
{
  "prompt": "Create a blue circle animation.",
  "model": "llama-3.3-70b-versatile"
}
Test with curl:
curl -X POST http://localhost:5000/v1/generate/code -H "Content-Type: application/json" -d @body.json
